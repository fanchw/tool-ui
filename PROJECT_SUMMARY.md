# 前端工具集项目 - 设计总结

## 📊 项目概览

**项目名称**: Developer Tools UI (开发者工具集)  
**项目类型**: 前端 Web 应用  
**技术栈**: React + TypeScript + Vite + shadcn/ui  
**预计工期**: 4-6 周  
**目标用户**: 前端开发者、全栈开发者

---

## 🎯 核心价值

### 为什么要做这个项目？

1. **提高开发效率**: 集成常用工具，无需在多个网站间切换
2. **隐私保护**: 所有操作在本地完成，不上传数据
3. **离线可用**: 支持 PWA，无网络也能使用
4. **开源免费**: 完全开源，可自由定制和部署
5. **现代化体验**: 优秀的 UI/UX 设计，流畅的交互

### 技术选型对比

| 技术 | 选择 | 替代方案 | 选择理由 |
|------|------|----------|----------|
| 框架 | React | Vue, Svelte | 生态成熟，社区活跃 |

| 构建工具 | Vite | Webpack, Parcel | 极快的开发体验 |

| CSS | Tailwind CSS | CSS Modules, Styled Components | 开发效率高，体积小 |
| 路由 | React Router | TanStack Router | 成熟稳定，文档完善 |
| 状态管理 | Zustand | Redux, Jotai | 简单易用，体积小 |
| 加密库 | crypto-js | node-forge | 浏览器兼容性好 |
| 日期库 | date-fns | moment.js, dayjs | 模块化，Tree-shaking 友好 |

## 🔧 核心功能详解

### 1. 加密解密工具

**功能清单**:
- ✅ AES 加密/解密 (CBC, ECB, CTR 模式)
- ✅ RSA 加密/解密 (公钥/私钥生成)
- ✅ DES/3DES 加密/解密
- ✅ 哈希生成 (MD5, SHA-1, SHA-256, SHA-512)
- ✅ HMAC 生成
- ✅ 密钥生成器

**技术实现**:
```typescript
import CryptoJS from 'crypto-js';

// AES 加密
const encrypted = CryptoJS.AES.encrypt(
  plaintext, 
  key, 
  { mode: CryptoJS.mode.CBC }
).toString();

// SHA-256 哈希
const hash = CryptoJS.SHA256(message).toString();
```

### 2. JSON 工具

**功能清单**:
- ✅ JSON 格式化 (美化)
- ✅ JSON 压缩 (最小化)
- ✅ JSON 转义/去转义
- ✅ JSON 验证 (语法检查)
- ✅ JSON 路径查询

**技术实现**:
```typescript
// 格式化
const formatted = JSON.stringify(JSON.parse(input), null, 2);

// 压缩
const compressed = JSON.stringify(JSON.parse(input));

// 验证
try {
  JSON.parse(input);
  return { valid: true };
} catch (error) {
  return { valid: false, error: error.message };
}
```

### 3. 时间戳工具

**功能清单**:
- ✅ 时间戳 → 日期时间
- ✅ 日期时间 → 时间戳
- ✅ 支持秒/毫秒/微秒
- ✅ 时区转换
- ✅ 相对时间计算

**技术实现**:
```typescript
import { format, parseISO } from 'date-fns';

// 时间戳转日期
const date = new Date(timestamp * 1000);
const formatted = format(date, 'yyyy-MM-dd HH:mm:ss');

// 日期转时间戳
const timestamp = Math.floor(date.getTime() / 1000);
```
